# Chapter 2 Data Pack: Skills Lab 2A Files

Starter files for Skills Lab 2A: Build the Club's Recycling Guide.
Students copy text out of the content file and repair the broken
page. They never retype provided content.

## Contents

| File | Type | Used in |
| ---- | ---- | ------- |
| `recycling-guide-content.txt` | Plain text page content with [BRACKET] labels | Parts 1-2 |
| `broken-contact.html` | HTML page with planted markup errors | Part 3 |
| `skills-lab-2a-answers.txt` | Plain text answer file (travels in the submission folder) | Parts 1-3, Questions & Analysis |

## File details

**recycling-guide-content.txt** holds every piece of visible text on
the finished `recycling-guide.html` page: the tab title, headings,
paragraphs, list items, link text, and addresses. Labels in
[BRACKETS] describe each piece so students can match content to the
right element. Students copy the text and type only the markup.

**broken-contact.html** is the club's contact page with exactly
four planted markup errors for the Part 3 validator repair task.
One defect can cascade into several messages, so the W3C
validator reports exactly 8 messages on the file as shipped. Students copy it into their lab folder as `contact.html`, run
it through the W3C validator, and fix every error. The repaired page
validates with zero messages.

**skills-lab-2a-answers.txt** holds numbered answer spaces for the
lab's written responses, including the Part 3 fix log and the
Questions & Analysis section. It travels inside the submission
folder unrenamed.
