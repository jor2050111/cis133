How This Bench Works

Open this page beside query-playground.css and drag the window edge slowly. Watch three things: the page background, the card row, and the banner title. Every change you see is written in the stylesheet, below the base rules it adjusts.

Keep DevTools open while you drag. The width readout in the top corner of the page area tells you the exact window width, so you can match what you see to the numbers in each query.
