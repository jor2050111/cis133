The px and rem Demo

Paragraph one is set at font-size: 16px.

Paragraph two is set at font-size: 1rem.

Paragraph three is set at font-size: 1.5rem.

Run the Experiment

  1. Open your browser's settings and search for font size. In Chrome and Edge the control lives under Appearance. In Firefox it lives under General, in the Fonts section.
  2. Raise the default font size from Medium (16) to a larger value, then return to this page.
  3. Compare the three paragraphs against what you predicted.
  4. Set the default back to Medium (16) when you finish.
