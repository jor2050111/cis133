# CIS133: Internet/Web Development Level I

Welcome to the CIS133 textbook. Build, style, and publish accessible multi-page websites with HTML and CSS, from your first page to a live site.

## About This Textbook

This textbook is designed for **community college students in their first web development course**. You need no prior experience. Every chapter starts from zero and builds on the chapters before it.

## How This Textbook Is Organized

The textbook is divided into **4 parts** spanning **12 chapters**, each designed to be completed in one week.

### Part I: Web Foundations (Chapters 1-3)

Learn how the Internet and the web work, then write your first HTML pages. By the end of Part I you can build a valid page with semantic structure, working links, and accessible images.

### Part II: Styling with CSS (Chapters 4-6)

Connect stylesheets to your pages and take control of color, spacing, and type. You finish this part with the box model and Flexbox skills to arrange a full page layout.

### Part III: Design for Every User (Chapters 7-9)

Plan sites with sitemaps and wireframes, adapt layouts to every screen size with media queries, and apply accessibility practices so every visitor can use your work.

### Part IV: Building Complete Sites (Chapters 10-12)

Present data with tables, collect input with forms, and preview what scripting adds to a site. The course ends where the web begins: you publish a complete multi-page website.

## Technology Stack

| Tool | Purpose |
|------|---------|
| **HTML5** | Structure page content with semantic markup |
| **CSS3** | Style color, spacing, type, and layout |
| **VS Code** | Write, organize, and manage your code files |
| **Browser DevTools** | Inspect pages, test layouts, and debug styles |
