# Skills Lab Rubric

Every Skills Lab in this book is graded with this one rubric: four
criteria, 4 points each, 16 points total. The criteria split evenly
between technical execution (Code Accuracy and Efficiency, Output
Quality) and transferable skills (Documentation and Code Comments, plus
Analysis, Interpretation, and Response to QUESTION(s)). Learn this
rubric once and you know what every lab expects of you.

## How to read the four criteria

* **Code Accuracy and Efficiency:** does the work run and produce
  correct, efficient results in the languages and tools this course
  teaches?
* **Output Quality:** are the outputs (charts, tables, pages, printed
  results) clear, well-labeled, and appropriate for the task?
* **Documentation and Code Comments:** could a colleague read the work,
  understand why each choice was made, and build on it?
* **Analysis, Interpretation, and Response to QUESTION(s):** is the
  reasoning sound, is every interpretation supported by evidence, and
  are the lab's QUESTION(s) answered fully?

The first two criteria grade what your code does. The last two grade
how well you communicate your thinking. Both halves carry equal weight
because both are the job: an analysis nobody can follow, verify, or
build on has not finished its work.
